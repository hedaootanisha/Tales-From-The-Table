This is a web application that shows a list of cafés I have visited. It is built using HTML, CSS, JavaScript, and Node.js.

The main idea of this project is to help friends and family quickly find good cafés and get the correct directions easily. Users can view café details and go to their Google pages for more information.

The app has a simple design and easy navigation. It also allows users to give feedback and share their experiences, so others can make better choices.
